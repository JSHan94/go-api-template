# initia-apis
Collection of RESTful APIs provided by Initia nodes

# Prerequisites

- Go 1.19+

# Project setup

## 1. Clone the project

```bash
git clone https://github.com/initia-labs/initia-apis.git
cd initia-apis
```

## 2. Configure Environment Variables

## 3. Run the project

### Run with binary
```
make all
```

### Run with Docker
```bash
make docker
```
